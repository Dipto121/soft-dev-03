About

The new management system will allow the user to interact with the admin directly through the use of this application to address his/her queries and this will also enable the admin to make the functionalities faster.

Proposed System:

  1. The Mess Management System helps the user to access all the functionalities of
    the mess without having to visit the mess physically and to apply for leave.
  2. It enables the admin to view the inventory and access guest details.
    This application is free of cost for the users.
  3. ndividuals who wish to use any functionality of the mess can simply
    log in to the app and have everything on their fingertips.
  4. It uses the internet to update all user queries and put it across to the admin.
  5. Using the information provided by all the users the admin can take
    decisions and the inventory for the mess can be managed.
